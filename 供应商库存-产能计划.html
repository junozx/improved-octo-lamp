计算逻辑：

1、已过去日期的供需分析:

总需求 = 实际订单量 + 安全库存量

   实际订单量:已经确认的客户订单

   安全库存量:根据库存策略设定的安全库存水平

总供应 = 销售出库量 + 仓库入库量

   销售出库:实际发货给客户的数量

   仓库入库:生产完工入库+外购入库数量


供需达成率 = (销售出库量 + 仓库入库量)/(实际订单量 + 安全库存量) × 100%


2、未来日期的供需预测:

总需求 = 销售计划量 + 安全库存量

   销售计划:包含确认订单和预测订单

   安全库存:预计所需的安全库存水平

总供应 = 期初库存 + Σ(未来生产计划)

   期初库存:当前实际库存量

   未来生产计划:包含已排产和计划排产的数量


假设今天是6月1日,计划周期3个月:

A. 5月(已过去):

总需求:实际订单1000 + 安全库存100 = 1100

总供应:出库900 + 入库200 = 1100

达成率:100%

B. 6月(当月):

总需求:确认订单800 + 预测200 + 安全库存100 = 1100

总供应:期初库存200 + 生产计划900 = 1100

预计达成率:100%

C. 7月(未来):

总需求:确认订单500 + 预测400 + 安全库存100 = 1000

总供应:预计结转库存200 + 生产计划800 = 1000

预计达成率:100%
